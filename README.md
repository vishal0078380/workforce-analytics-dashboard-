# Workforce Analytics & Org Intelligence Dashboard

![CI/CD Pipeline](https://github.com/vishal0078380/workforce-analytics-dashboard/actions/workflows/deploy.yml/badge.svg)

A cloud-native HR analytics platform built on AWS.

## Tech Stack

- **Frontend:** React.js hosted on Amazon S3
- **Auth:** Amazon Cognito
- **APIs:** AWS Lambda + API Gateway
- **Database:** Amazon DynamoDB
- **Monitoring:** Amazon CloudWatch
- **CI/CD:** GitHub Actions

## Live URL

http://workforce-dashboard-bucket.s3-website.ap-south-1.amazonaws.com

## Features

- Department Headcount Analytics
- Recruitment Funnel Tracking
- Leave Utilization by Department
- Attrition Risk Scoring
- Interactive Org Chart (D3.js)
- System Health Monitoring
